Yalan Haber Tespit Modeli
Dijital dezenformasyonla mücadele etmek amacıyla geliştirdiğim makine öğrenmesi tabanlı içerik analiz projesidir.

✨ Öne Çıkan Özellikler
NLP Analizi: Haber metinlerini doğal dil işleme teknikleriyle temizler ve sınıflandırmaya hazır hale getirir.

Model Eğitimi: Makine öğrenmesi algoritmalarıyla haber metinlerini analiz ederek "Gerçek" veya "Sahte" olarak etiketler.

Web Arayüzü: Streamlit altyapısı ile çalışan, kullanıcıların doğrudan haber metni girip anlık sonuç alabildiği interaktif bir yapı sunar.

Kullanılan Teknolojiler
Programlama: Python

Veri Bilimi & ML: Pandas, NumPy, Scikit-learn

Arayüz & Dağıtım: Streamlit
